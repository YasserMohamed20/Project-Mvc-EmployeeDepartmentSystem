# 🚀 Employee Department Management System

---

# 📖 Overview

Employee Department Management System is an ASP.NET Core MVC application that allows users to efficiently manage employees and departments through a clean and responsive user interface.

The project demonstrates practical implementation of:

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views
- CRUD Operations
- Dependency Injection
- LINQ
- Model Validation
- Bootstrap UI

---

# ✨ Features

## 👨 Employee Management

- ➕ Add Employee
- ✏️ Edit Employee
- ❌ Delete Employee
- 🔍 View Employee Details
- 📋 Display Employees List
- 🖼 Upload Employee Image
- 🏢 Assign Employee to Department

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/YasserMohamed20/Project-Mvc-EmployeeDepartmentSystem.git
```

## Navigate to Project

```bash
cd Project-Mvc-EmployeeDepartmentSystem
```

## Restore Packages

```bash
dotnet restore
```

## Update Database

```bash
dotnet ef database update
```

## Run Project

```bash
dotnet run
```
